# Icône de l'application OCP Excursions

## Fichiers

- **`app_icon.png`** : icône au format PNG (utilisable pour un favicon ou un raccourci).

## Obtenir un fichier .ico (pour Windows)

Windows utilise le format **.ico** pour les raccourcis.

1. **En ligne** : convertissez `app_icon.png` en .ico sur un site comme [convertio.co](https://convertio.co/fr/png-ico/) ou [icoconvert.com](https://icoconvert.com/). Téléchargez **app_icon.ico**.
2. Placez **app_icon.ico** dans ce dossier **`icons/`** si vous en avez besoin (par ex. pour un raccourci vers http://localhost:3000).

## Utilisation

L’icône peut servir pour un **raccourci sur le Bureau** vers la plateforme (http://localhost:3000) : créez un raccourci, puis en Propriétés → Changer l’icône, sélectionnez **`icons/app_icon.ico`**.

La plateforme s’exécute en **local** (backend + frontend) ; voir le **README.md** à la racine du projet pour le démarrage.
